# Source-Code-of-Affine-Line-Universal-Cycle

This repository contains the Python implementation used in my Master's thesis, Construction of Universal Cycles for Affine Lines over Finite Fields. For ease of use, the code is fully compatible with Google Colab for direct execution.

This is an interactive program that allows users to specify the exact size of a finite field and the dimension of the affine space. Following the methodology established in the thesis, the program will computationally construct a valid universal cycle of affine lines corresponding to the provided parameters.

Furthermore, the implementation includes a built-in validator specifically designed to rigorously verify whether a generated sequence strictly satisfies the mathematical requirements of a universal cycle.
